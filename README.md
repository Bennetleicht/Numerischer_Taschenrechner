# Numerischer_Taschenrechner
Dieses Python-Tool visualisiert numerische Algorithmen für Lehre und Selbststudium. Es macht Iterationen und Konvergenz bei Problemtypen wie Nullstellen oder DGL grafisch sichtbar. Nutzer können Parameter live anpassen und Verfahren direkt vergleichen. Die modulare Architektur bietet eine intuitive GUI für komplexe Berechnungen.
